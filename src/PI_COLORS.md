# Pi TUI Theme Colors

Named colors available from `@mariozechner/pi-tui`. Only these names are valid — raw CSS/hex values throw a runtime error.

| Color | Actual appearance | Current usage |
|---|---|---|
| `text` | Default foreground | Main data values (prices, numbers) |
| `dim` | Dark grey | Labels, separators, box borders |
| `muted` | Lighter grey | Gondolier role color |
| `accent` | Blue/cyan | Wallet name, size label, epoch countdown, gauge fill |
| `success` | Green | Uptick, positive changes |
| `error` | Red | Downtick, negative changes, Mercenary role |
| `warning` | Yellow | Patrician role |
| `syntaxType` | Teal | Merchant role, locked gauge |
| `syntaxKeyword` | Blue | Section headers (STAKING, DIEM, SYSTEM, etc.), VVV/DIEM price labels |
